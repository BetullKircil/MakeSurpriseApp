# Welcome to Your Expo App 👋

Bu proje, [`create-expo-app`](https://www.npmjs.com/package/create-expo-app) kullanılarak oluşturulmuş bir [Expo](https://expo.dev) projesidir.

## 🚀 Başlangıç

Projeyi çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

### 1️⃣ Bağımlılıkları Yükleyin

```bash
npm install
```

### 2️⃣ Uygulamayı Başlatın

```bash
npx expo start
```

Bu komut ile aşağıdaki seçeneklerden biriyle uygulamayı başlatabilirsiniz:

- [Development Build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android Emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS Simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go): Expo projelerini test etmek için hafif bir geliştirme ortamı sağlar.

Projeye başlamadan önce **app** klasörü içindeki dosyaları düzenleyerek geliştirmeye başlayabilirsiniz. Bu proje, [dosya tabanlı yönlendirme](https://docs.expo.dev/router/introduction/) sistemini kullanmaktadır.

---

## 📌 Yeni Bir Proje Başlatın

Projeyi sıfırdan başlatmak isterseniz aşağıdaki komutu çalıştırabilirsiniz:

```bash
npm run reset-project
```

Bu komut, mevcut başlangıç kodlarını **app-example** klasörüne taşıyacak ve yeni bir **app** klasörü oluşturarak temiz bir başlangıç yapmanızı sağlayacaktır.
